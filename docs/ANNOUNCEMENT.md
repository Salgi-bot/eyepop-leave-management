## 2. 연차 신청 방법

### 2-1. 메인 화면 → **연차 신청** 클릭

### 2-2. 본인 정보 입력
- 이름, 회사 이메일

### 2-3. 휴가 종류 선택

| 종류 | 일수 |
|---|---|
| **연차** | 1일 (8시간) |
| **3/4차** | 0.75일 (6시간) |
| **오전반차** | 0.5일 (4시간, 09:00~13:00) |
| **오후반차** | 0.5일 (4시간, 14:00~18:00) |
| **반반차** | 0.25일 (2시간) |

### 2-4. 날짜 선택 + 사유 입력
- 사유는 **2자 이상 필수**
- 부적절한 표현 자동 차단

### 2-5. 팀장 구두보고 확인 체크 + 신청

### 2-6. 신청 완료
- 본인 + 김은주 차장(경영기획실)에게 자동 메일 발송
- 일반 팀원의 경우 팀장도 CC

---

## 3. 신청 후 흐름

```
[직원 신청]
   ↓ 자동 알림 메일
[경영기획실 김은주 차장 검토]
   ↓ 승인 또는 반려
[직원에게 결과 메일]
   ↓
[직원이 메일의 "수신 확인" 버튼 클릭]
   → 회사 도달 사실 기록 완료
```

**메일 받으시면 반드시 "수신 확인" 버튼을 1회 클릭**해주세요. 도달 사실 증명 용도입니다.

## 4. 자주 묻는 질문

### Q1. 잔여 연차는 어디서 확인하나요?
신청 페이지에서 본인 이름·이메일 입력 시 자동 표시됩니다.

### Q2. 잘못 신청했어요
경영기획실에 연락 → 반려 처리 → 다시 신청해주세요.

### Q3. 반차 시간을 다르게 쓰고 싶어요 (예: 13:30~17:30)
**반반차** 두 번 사용 또는 **오후반차** 신청 + 사유에 "13:30~17:30 사용" 명기.

### Q4. 메일이 안 와요
스팸함 확인 → 그래도 없으면 경영기획실 문의.

### Q5. SECOM 출퇴근과 연차가 자동 대조됩니다
- **9시 이전 / 9:30 이내 출근 → 정시 출근으로 인정**
- **18시 이후 퇴근 → 정시 퇴근으로 인정**
- 8시간 이상 근무는 모두 정상 처리
- 미신청 결근은 자동 검출됨

### Q6. 임원도 신청해야 하나요?
네, 임원도 신청 대상입니다 (김홍정 부사장만 SECOM 대조 미적용).
