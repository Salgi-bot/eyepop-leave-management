## 2. 연차 신청 방법

### 2-1. 위 직원용 주소 접속

### 2-2. 본인 정보 입력
- 이름, 회사 이메일

### 2-3. 휴가 종류 선택

| 종류 | 일수 | 시간 |
|---|---|---|
| **연차** | 1일 | 8시간 |
| **3/4차** | 0.75일 | 6시간 |
| **반차** | 0.5일 | 4시간 (시간 직접 입력) |
| **반반차** | 0.25일 | 2시간 (시간 직접 입력) |

> 💡 **반차는 4시간 한정으로 사용 시간을 자유롭게 지정**할 수 있습니다.
> · 오전 반차 표준: **09:00~14:00** (점심 12:00~13:00 제외 = 4시간)
> · 오후 반차 표준: **14:00~18:00**
> · 그 외 시간 가능 (예: 10:00~15:00, 점심 1시간 제외 4시간)

### 2-4. 날짜 선택 + 사유 입력
- 사유는 **2자 이상 필수**
- 부적절한 표현 자동 차단

### 2-5. 팀장 구두보고 확인 체크 + 신청

### 2-6. 신청 완료
- 본인 + 경영기획실 김은주 차장에게 자동 메일 발송
- 일반 팀원의 경우 팀장도 CC

## 3. 신청 후 흐름

```
[직원 신청]
   ↓ 자동 승인 + 알림 메일
[직원에게 승인 메일 도착]
   ↓
[메일의 "수신 확인" 버튼 1회 클릭]
   → 회사 도달 사실 기록 완료
```

**메일 받으시면 반드시 "수신 확인" 버튼을 1회 클릭**해주세요. 도달 사실 증명 용도입니다.

## 4. 자주 묻는 질문

### Q1. 잔여 연차는 어디서 확인하나요?
**신청 후 받는 자동 메일**과 **신청 화면의 결과**에 잔여 연차가 표시됩니다. 메일을 보관해두시면 됩니다.

### Q2. 잘못 신청했어요 (철회·취소)
신청 메일 본문의 **[❌ 신청 철회하기]** 버튼을 클릭하면 즉시 철회됩니다.
- 본인·관리자·팀장에게 자동 통보 메일 발송
- 잔여 연차 자동 재계산
- 유효기간: 신청 후 30일

> 30일이 지났거나 메일을 잃어버린 경우 김은주 차장(eunju@eyepopeng.com)에게 연락해주세요.

### Q3. 반차 시간을 다르게 쓰고 싶어요 (예: 10:00~14:00)
**반차** 선택 시 시간 입력 칸에 직접 4시간 범위를 기재합니다 (예: "09:00~14:00", "14:00~18:00", "10:00~15:00").

### Q4. 메일이 안 와요
스팸함 확인 → 그래도 없으면 **김은주 차장에게 문의**.

### Q5. SECOM 출퇴근과 연차가 자동 대조됩니다
- **9시 이전 / 9:30 이내 출근 → 정시 출근으로 인정**
- **18시 이후 퇴근 → 정시 퇴근으로 인정**
- 8시간 이상 근무는 모두 정상 처리
- 미신청 결근은 자동 검출됨

### Q6. 임원도 신청해야 하나요?
**네, 전 임직원 대상**입니다.
