📢 EYEPOP 연차관리 시스템 도입 안내

📅 시행일: 2026-05-01
👥 전 임직원 대상

연차 신청·승인 절차를 효율화하기 위해 EYEPOP 연차관리 시스템을 도입합니다.

⚠️ 5월 1일부터 시행됩니다.
⚠️ 이번 달(4월) 연차 사용 예정자는 미리 신청해주세요.

━━━━━━━━━━━━━━━━━━

🔗 시스템 접속

leave.eyepopeng.com

주소만 입력하면 바로 연차 신청 화면이 열립니다. 즐겨찾기에 추가하시면 편합니다.

━━━━━━━━━━━━━━━━━━

📋 휴가 종류

✓ 연차 — 1일 (8시간)
✓ 3/4차 — 0.75일 (6시간)
✓ 반차 — 0.5일 (4시간, 시간 직접 입력)
✓ 반반차 — 0.25일 (2시간, 시간 직접 입력)

💡 반차 시간 자유롭게 지정 가능
· 오전 반차 표준: 09:00~14:00 (점심 1시간 제외 = 4시간 사용)
· 오후 반차 표준: 14:00~18:00
· 그 외 가능 (예: 10:00~15:00)

━━━━━━━━━━━━━━━━━━

📝 신청 방법

1️⃣ 시스템 접속 → [📝 연차 신청하기] 클릭
2️⃣ 이름·이메일·날짜·사유 입력 → 휴가 종류 선택
3️⃣ 팀장 구두보고 확인 체크 → [신청 제출]
4️⃣ 자동 메일 도착 (본인 + 김은주 차장)
5️⃣ 메일의 [📩 수신 확인하기] 버튼 1회 클릭 → 회사 도달 기록 완료

━━━━━━━━━━━━━━━━━━

❌ 잘못 신청한 경우 (철회·취소)

신청 메일 본문의 [❌ 신청 철회하기] 버튼을 클릭하면 즉시 철회됩니다.
· 본인·관리자·팀장에게 자동 통보
· 잔여 연차 자동 재계산
· 유효기간: 신청 후 30일

30일이 지났거나 메일을 잃어버린 경우 김은주 차장(eunju@eyepopeng.com)에게 연락 부탁드립니다.

━━━━━━━━━━━━━━━━━━

❓ 자주 묻는 질문

Q. 잔여 연차는 어디서 확인?
→ 신청 후 받는 자동 메일과 화면 결과에 표시됩니다. 메일을 보관해두시면 됩니다.

Q. 메일이 안 와요
→ 스팸함 먼저 확인 → 그래도 없으면 김은주 차장에게 문의

Q. 출퇴근 기록과 대조되나요?
→ 매월 초 SECOM 기록과 대조하며, 이상 발견 시 재확인 절차를 진행합니다.

Q. 임원도 신청해야 하나요?
→ 네, 전 임직원 대상입니다.
