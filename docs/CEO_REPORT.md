## 1. 보고 요지

기존 연차 신청·승인 프로세스의 비효율을 개선하고 근로기준법상 사용촉진 통지 등 법정 의무 이행의 정확성을 확보하기 위해, **사내 자체 개발한 EYEPOP 연차관리 시스템**의 도입을 완료하고 **2026년 5월 1일부**로 본격 운영을 시작하고자 합니다.

---

## 2. 도입 배경

기존 연차 운영 방식의 한계점:

| 구분 | 기존 방식 | 문제점 |
|---|---|---|
| 신청 절차 | 수기 양식·이메일 분산 | 누락·소실 위험, 처리 지연 |
| 잔여 연차 확인 | 수동 집계 | 직원·관리자 모두 실시간 파악 불가 |
| 출퇴근 기록 대조 | 별도 수작업 | SECOM 기록과 연차 데이터 분리 |
| 사용촉진 통지 (근기법 제61조) | 수동 발송 | 누락 시 미사용 연차 보상 의무 발생 |
| 감사 추적성 | 미흡 | 신청·승인·변경 이력 추적 어려움 |

→ 이를 해결하기 위해 **사내 자체 개발 + 클라우드 운영(외부 라이선스 비용 없음)** 방식으로 시스템을 구축하였습니다.

## 3. 시스템 주요 기능

### 3-1. 직원 측 기능
- 웹 기반 신청 (PC·모바일 동일 접속): https://leave.eyepopeng.com
- 휴가 종류 5종: 연차·3/4차·반차(시간 직접 입력)·반반차·없음
- **자동 잔여 연차 표시** (신청 시점 즉시)
- **메일 [철회] 1클릭** (잘못 신청 시 직원이 직접 철회, 별도 인증 불필요)

### 3-2. 관리자 측 기능 (경영기획실)
- 신청·승인·반려·수정·삭제 통합 관리 화면
- **자동 승인 모드** (관리자 부담 최소화)
- 직원 명단 엑셀 일괄 등록
- SECOM 출퇴근 기록 자동 대조
- 모든 액션에 대한 본인·관리자·팀장 자동 통지 메일

### 3-3. 법정 의무 자동화
- **연차 사용촉진 통지 자동 발송** (7월·10월, 근로기준법 제61조)
- 통지 발송 이력 자동 보관 (감사 대응용)
- 메일 수신 확인 토큰으로 도달 사실 증명

## 4. 도입 효과 (기대)

### 4-1. 정량 효과 (업무·시간·비용)

| 항목 | 기존 | 도입 후 | 절감 |
|---|---|---|---|
| 신청 1건 처리 시간 | 약 5분 (수기·이메일) | 약 1분 (자동 승인) | **80%** |
| 월 평균 관리자 업무 시간 (가정 30건/월) | 약 5시간 | 약 1시간 | **80%** |
| 사용촉진 통지 누락 위험 | 수동 운영 시 존재 | 자동화로 0% | **법적 리스크 제거** |
| 외부 솔루션 도입비 (참고) | 연 200~500만 원 (SaaS) | **0원** (자체 개발) | — |

### 4-2. 종이·출력 절감 (ESG 효과)

| 항목 | 기존 (수기 운영) | 도입 후 (디지털) | 효과 |
|---|---|---|---|
| 신청서·승인서·통지서 출력 | 1건당 3~4매 | **0매** (전자 처리) | **100%** |
| 월 종이 사용량 (30건/월 가정) | 약 90~120매 | 0매 | 연 1,000매 이상 절감 |
| 종이·토너·결재 사본·보관 자재 등 직간접 비용 | — | — | **연 약 100만 원 이상 절감** |
| 보관 공간 | 캐비닛 1칸 (3년 누적) | **0매** (디지털 영구 보관) | 물리 공간 회수 |
| 분실·훼손 위험 | 종이 소실·인쇄 오류 가능 | 0% (영구 보관·자동 백업) | 감사 대응력 향상 |

> ※ 직간접 비용 = 종이·토너 직접 비용 + 결재용 사본·재출력 + 보관 자재(바인더·라벨) + 정리·검색 인건비 + 캐비닛 점유 공간 비용 합산 추정치

**부가 효과**:
- 결재·서명 위한 출력 절차 제거 → 결재 소요시간 단축
- 캐비닛·서류함 공간 회수
- 친환경 경영(ESG) 지표 부합 — 탄소 배출·종이 사용 감축
- 재택·외근 직원 신청 시 출력·스캔 부담 0

### 4-3. 정성 효과
- 직원 편의성 향상 (모바일 즉시 신청·메일 1클릭 철회)
- 경영기획실 업무 부담 감소 → 본연 업무 집중 가능
- 연차 사용 현황 실시간 가시성 (임원 회의 활용 가능)
- 감사 추적성 강화 (모든 작업 영구 보관)
- SECOM 출퇴근 자동 대조로 출결 관리 정확도 향상
