| | |
|---|---|

| **발신** | 경영기획실 |

---

## □ 보고 요지

연차 신청·승인 절차의 효율화 및 근로기준법상 사용촉진 통지 의무 이행의 정확성 확보를 위해, **사내 자체 개발한 EYEPOP 연차관리 시스템**의 도입을 완료하고 **2026년 5월 1일부**로 본격 운영을 시작하고자 함.

---

## □ 추진 배경

- 수기·이메일 분산으로 인한 누락·소실 위험
- 잔여 연차 실시간 파악 불가
- SECOM 출퇴근 기록과 연차 데이터 분리
- 연차 사용촉진 통지(근로기준법 §61) 누락 시 미사용 연차 보상 의무 발생

---

## □ 시스템 주요 기능

- 웹 기반 신청 (PC·모바일 동일): **leave.eyepopeng.com**
- 자동 승인 모드 + 메일 [철회] 1클릭 (별도 인증 불필요)
- SECOM 출퇴근 자동 대조
- 사용촉진 통지 자동 발송 (7월·10월)
- 모든 작업 영구 보관 (감사 추적성 확보)

---

## □ 기대 효과

| 항목 | 기존 | 도입 후 |
|---|---|---|
| 신청 1건 처리 시간 | 약 5분 | 약 1분 (80%↓) |
| 월 관리자 업무 시간 (30건/월 가정) | 약 5시간 | 약 1시간 (80%↓) |
| 사용촉진 통지 누락 위험 | 존재 | 0% (자동화) |
| 외부 SaaS 도입비 (참고) | 연 200~500만 원 | 0원 (자체 개발) |
| 종이·인쇄·보관 직간접 비용 | 연 약 100만 원 이상 | 0원 |

**부가 효과**: ESG 경영 부합(친환경·탄소 절감) · 감사 추적성 강화 · 직원 편의성 향상
