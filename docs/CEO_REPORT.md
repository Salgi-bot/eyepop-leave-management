## 1. 추진 배경

기존 연차 신청·승인 프로세스의 비효율 및 근로기준법상 사용촉진 통지 의무 이행의 정확성 확보를 위해, 사내 자체 개발 시스템을 도입하여 **2026년 5월 1일부**로 본격 운영을 개시함.

### 가. 기존 운영 방식의 한계

- 수기·이메일 분산으로 인한 누락·소실 위험
- 잔여 연차 실시간 파악 불가
- SECOM 출퇴근 기록과 연차 데이터 분리
- 사용촉진 통지(근로기준법 §61) 누락 시 미사용 연차 보상 의무 발생
- 신청·승인·변경 이력 추적성 미흡

## 2. 시스템 주요 기능

### 가. 직원 측 기능

- 웹 기반 신청 (PC·모바일 동일): leave.eyepopeng.com
- 휴가 종류 5종: 연차·3/4차·반차(시간 직접 입력)·반반차·없음
- 자동 잔여 연차 표시 (신청 시점 즉시 메일 안내)
- 메일 [철회] 버튼 1클릭 (별도 인증 불필요)

### 나. 관리자 측 기능 (경영기획실)

- 신청·승인·반려·수정·삭제 통합 관리 화면

- 직원 명단 엑셀 일괄 등록
- SECOM 출퇴근 자동 대조
- 모든 액션에 본인·관리자·팀장 자동 통지 메일

### 다. 법정 의무 자동화

- 연차 사용촉진 통지 자동 발송 (7월·10월, 근로기준법 §61)
- 통지 발송 이력 자동 보관 (감사 대응)
